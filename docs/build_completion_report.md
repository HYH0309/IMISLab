## 🚀 部署准备

### 前端部署

1. **静态文件服务器**: 将 `frontend/dist/` 目录内容部署到Nginx/Apache
2. **CDN部署**: 可直接上传到CDN服务
3. **Docker部署**: 可使用Nginx镜像承载静态文件

### 后端部署  

1. **Linux服务器**: 直接运行 `./IMISLab_backend`
2. **系统服务**: 配置为systemd服务
3. **Docker部署**: 基于Alpine Linux镜像

### 环境配置

- **Redis**: 确保Redis服务运行 (端口6379)
- **数据库**: MySQL/MariaDB配置
- **端口**: 后端默认3344端口
- **环境变量**: 检查.env文件配置
